# rpm (Red hat package manager)

### Note: We are installing package on debian based linux and each OS release are different

---

### How to install files on OS?
**You can use `rpm` command to install your files on your `OS` manually**

```bash
rpm -ivh wget-1.14-18.e17_6.1.x86_64.rpm
```

### Note: `-vh` is for more information diuring installation and it's not important you can only use `-i`

### Note: It's possible that package you are trying to install have some dependency to install and you don't have that dependency on your system and you need to first install dependencys and after that again try to install first package you where tried to install

---

### How to install package files on server?
**You can install your file with their link like below**

```bash
wget http://archive.ubuntu.com/ubuntu/pool/man/z/zip/zip_3.0-12build2_amd64.rpm
```

---

### Note: It's possible we don't have wget in red hat base OS like centos for this problem we install wget first with rpm and then get our other packages

---

### If we don't have wget how we can install wget on OS?
**First you need to install `.rpm` of your command and then transfer it with `scp` for linux base `os` or `winscp` for windows on your server**

---

### How to undrestand the version of package?
**You can use `-q` to undrestand the version of that package you have installed**

```bash
rpm -q zip
```

---

### How to know where your .rpm file copied files?

```bash
rpm -ql zip
```

---

### If you find a route how you can know which package made it?

```bash
rpm -qf /usr/share/doc/zip-3.0/algorith.txt
```

---

### How to give full information about a package?
**You can inform about some information about your installed package look like what is package name and when installed and ...**
```bash
rpm -qi zip
```

---

### How to inform information about .rpm package?

```bash
rpm -qip wget-1.14-18.e17_6.1.x86_64.rpm
```

### Note: You can run this command when you downloaded a `.rpm` file and you don't know information and you want to install it on server it's recomended to use this command before installation
---

### How to undrestand about dependencies of packages?

``` bash
rpm -qpR wget-1.14-18.e17_6.1.x86_64.rpm
```

---

### How to find a package configuration files?

```bash
rpm -qc zip
```

---

### How to know what is whole of packages we have installed on our OS

```bash
rpm -qa
```

---

### If your `.rpm` package have a new version you can upgrade it

```bash
rpm -Uvh wget-1.14-18.e17_6.1.x86_64.rpm
```

### Note: `Uvh` command first will check if new version is available it will upgrade if not it will install automaticlly it's like `ivh`

---

### If you installed a package and you need to remove that package `.rpm` file you can do it like below

```bash
rpm -e zip
```

### Note: In `-e` you need only give package name your OS will automaticlly find that package `.rpm` file and remove it

---

### You can give verbos from rpm command max 5 verbos
**In linux you can use maximum to 5 `-v` but your command must support it**

```bash
rpm -ev zip
```

```bash
rpm -evv zip
```

### Note: Each `-v` give you more informations
