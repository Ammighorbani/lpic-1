# base64

### You can encode your text with this algorithm and decode them again

---

### Encode

**Input**
```bash
echo "Ammighorbani" | base64
```

**Output**
```bash
QW1taWdob3JiYW5pCg==
```

### Decode

**Input**
```bash
echo "QW1taWdob3JiYW5pCg==" | base64 -d
```

**Output**
```bash
Ammighorbani
```
