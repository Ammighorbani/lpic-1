# ssh

### You can connect with ssh protocol to your server but in normal way you will use your username and password but you can do it without your password

---

### How to create ssh key?
**You can create `public` and `private` ssh key**
```bash
ssh-keygen
```

### Note: If you put your public ssh key on the other servers and have your private key on your system you can connect to that other servers with ssh key

---

### How to connect to the other servers?
**You must copy your public key and put it in `/root/.ssh/authorized_keys` file and you can put morethan one ssh public key**
