# vim

### You can use vim tutorial with `vimtutor` command

### You can make and edit your file with `vim` and vim is a code editor on linux
**You have 3 different important mode with `command mod`, `insert mode` and `replace mode`**

---

### What is `command mode`?
**In `command mode` you can use only vim customed command and you can't write or edit any thing**

### What is `insert mode`?
**In `insert mode` you only can delete or write and your vim custom command won't work**

### Note: whole of the time when you are using vim you are switching between command mode and insert mode

### Command mode short keys:

### Jumping

### You have 4 keys to move between texts one char

```
1. h: You can go left
2. j: You can go down
3. k: You can go up
4. l: You can go right
```

### You have some keys to move between texts one word

```
1. w: Move one word one word to start of word
2. e: Move one word one word to end of word
3. b: Come back one word one word
```

### You can use your short keys with numberX
**You can use numbers to cross your operation for examle if `w` move one word one word you can use `4w` to move 4 words**


### How to jump top, down and middle of screen?

```
1. SHIFT + h: Jump top of screen
2. SHIFT + l: Jump down of screen
3. SHIFT + m: Jump middle of screen
```

### How to jump start and end of file

```
1. gg: Jump start of file
2. SHIFT + g: Jump end of file
```

### How to jump end and start of line

```
1. $: Jump end of line
2. 0: Jump start of line
```

### How to move specefic line of file?

```
number of line + gg: Jump to specefic line
```

### Searching

### How to go `search mode`?
```bash
/
```

### How to search for specefic word?
```
/[pattern]
```

### Note: After you searched for pattern you can use `n` on your keyboard to next one and `SHIFT + n` on your keyboard to recent pattern you have in your file

### How to replace first pattern in that line your cursor on it?
```bash
:s/sshd/SSHD/
```

### Note: First place is for that pattern you want to search and second place is for that pattern you want to change it

### How to replace all patterns in that line your cursor on it?
```bash
:s/sshd/SSHD/g
```

### How to replace all pattern in whole of file?
```bash
:%s/sshd/SSHD/g
```

### How to replace all pattern in whole of file with confirmation pattern by pattern?
```bash
:%s/sshd/SSHD/gc
```

### How to replace all pattern in whole of file with confirmation pattern by pattern and case insensitive?
```bash
:%s/sshd/SSHD/gi
```

