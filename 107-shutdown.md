# shutdown

### You can turn off your system with `shutdown` command

---

### You can `turn off` your system
```bash
shutdown
```

---

### you can `reboot` your system
```bash
shutdown -r / --reboot
```

---

### you can `poweroff` your system
```bash
shutdown -P / --poweroff
```

---

### you can `halt` your system
```bash
shutdown -H / --halt
```

---

### You can cancel any `poweroff` or `shutdown` with
```bash
shutdown -c
```

---

### You can do `force` operation with `shutdown` command
```bash
shutdown -r now
```

---

### You can schedual specefic time for `turn off`
```bash
shutdown 15:45
```

---

### You can `turn off` specefic time later from now
```bash
shutdown -t 1000 +1000
```
