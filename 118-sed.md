# sed

### `sed` Command is look like find patter and replacement in `vim` but with out opening the files

```bash
sed 's/yes/no/g'
```

### How to write in file our change?
**You can use `-i` switch**

```bash
sed -i 's/yes/no/g'
```

### How to be case insensitive?

```bash
sed -i 's/yes/no/gi'
```

#### `s`: **For replacement**
#### `g`: **For doing global**
#### `i`: **For case insensitive**
