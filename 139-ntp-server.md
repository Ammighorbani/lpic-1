# NTP server (Network Time Protocol)

### You can create a server and configure time settings on it, after that you can tell to your other servers to ask time from your `NTP server`

---

### We have two sides of NTP server

#### `NTP client`: **Who ask time from other servers**
#### `NTP server`: **Who answer time to clients**

---

### How to see public NTP servers whole around the world?

**You can see public `NTP servers` whole around the world in `ntp.pool.org` website**
